    Batting Filter ガイド

    ⚾ 投手タイプ

    右投手・左投手別に 大谷翔平の打撃成績を確認できます。

    📊 打席結果

    HIT / HR / BB / SO など、 結果別に打席を絞り込みできます。

    🏟 チーム

    対戦チームごとの 打撃傾向を確認できます。

    👤 投手名

    特定投手との対戦成績を検索できます。
    例：今永昇太、Logan Webb、 Jacob deGromなど今季対戦した投手であれば 1文字づつ入力ごとに補完し候補表示します(サジェスト検索機能)

    🌀 球種

    Sweeper / Splitter / Four-Seam など、 球種別に打撃結果を絞り込みできます。

    🚀 球速レンジ

    指定した球速帯に対する 打撃結果のみを抽出します。
    ドラムロールまたは スライダーで調整できます。

    💡 検索のコツ

    「左投手 + Sweeper + 95mph以上」など、 条件を組み合わせることで 傾向分析ができます。
